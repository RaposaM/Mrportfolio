# Mitchell Raposa - Portfolio Website

A modern, responsive portfolio website built with React, TypeScript, and Tailwind CSS.

## Features

- **Multi-page navigation**: Home, About, Projects pages with React Router
- **Responsive design**: Optimized for mobile and desktop viewing
- **Interactive elements**: Hover effects and smooth transitions
- **Modern tech stack**: React 19, TypeScript, Tailwind CSS v4, Vite

## Pages

- **Home**: Hero section with personal introduction, about summary, featured projects, and experience timeline
- **About**: Detailed personal bio, skills showcase, and work experience
- **Projects**: Portfolio of design and development work

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS v4
- React Router v7
- Vite
- pnpm

## Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Start development server:
   ```bash
   pnpm dev
   ```

3. Build for production:
   ```bash
   pnpm build
   ```
